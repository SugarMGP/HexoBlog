编写一个控制器

什么是“控制器”（Controller）

在 Spring Boot 中，控制器是处理用户请求的入口。

你可以把它想象成一个网站或接口的“接待员”：
👉 用户发来请求，控制器接收、处理，然后返回响应。

@RestController 是什么

用来告诉 Spring 这个类是一个控制器（用来接收 Web 请求），并且返回的是 JSON 数据（而不是 HTML 页面）

@RequestMapping 是什么

告诉 Spring，这个类负责处理哪个 URL 请求

常用方法：加在类上，定义“公共前缀”
